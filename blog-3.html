#1 For help menu Answer Found: -h

#2 For stealth scan, Answer Found: -sS

#3 For UDP scan, Answer Found: -sU

#4 For Operating System Detection, Answer Found: -O

#5 For service version Detection, Answer Found: -sV

#6 For Verbosity Flag, Answer Found: -v

#7 For very verbose, Answer Found: -vv

#8 For saving output file in XML format, Answer Found: -oX

#9 For Aggressive Scan, Answer Found: -A

#10 For set the timing of max level, Answer Found: -T5

#11 For scanning a specific port, Answer Found: -p

#12 For scanning every port, Answer Found: -p-

#13 For enable using a script from the Nmap scripting engine, Answer Found: - -script

#14 For run all the scripts out of the vulnerability category, Answer Found: -- script vuln

#15 For not want to ping the host, Answer Found: -Pn
